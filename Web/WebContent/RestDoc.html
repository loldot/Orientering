O-løp API reference

orientering.no offers a REST-API which supports the following methods:

Only the url /jersey/articles is supported at the moment.

A table of the supported methods in orientering.nos API
http-verbparametersInput data typeReturnsReturn type
GETNoneNoneReturns all articles in the systemapplication/xml
GETidNoneReturns an article in the system with the given IDapplication/xml
POST*Noneapplication/xmlRedirects you to the new articletext/html
PUT*idapplication/xmlRedirects you to the updated articletext/html

* annotates that authentication header with a base64-encoded string of username:password is sent
